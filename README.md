# IEEE RAS Website

A responsive portfolio site for the IEEE Robotics & Automation Society (RAS) at the University of Jaffna. Built with React, Vite, and Tailwind CSS.

## Features
- Home page with hero video background
- News feed component displaying latest updates
- Events page showcasing upcoming activities
- Responsive design for desktop and mobile
- Easy to extend with additional sections

## Getting Started
```bash
npm install      # install dependencies
npm run dev      # start development server
npm run build    # create production build in ./dist
```

## Project Structure
- `src/` – React components and pages
- `src/assets/` – images, videos, and icons
- `public/` – static HTML template
- `vite.config.js` – Vite bundler configuration
- `tailwind.config.js` – Tailwind CSS setup
